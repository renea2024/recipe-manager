# 🍽️ Recipe Manager App

A web application to organize, search, and track your favorite recipes, built to practice JavaScript, HTML, and CSS while focusing on user-friendly design and functionality.

---

## 💡 Features
- Add, edit, and delete recipes easily
- Search recipes by name or ingredient
- Categorize recipes for quick access (e.g., Breakfast, Lunch, Dinner)
- Responsive design for mobile and desktop
- Persistent data using LocalStorage

---

## 🛠️ Technologies Used
- HTML5 – Semantic structure and accessibility
- CSS3 – Flexbox layout and responsive styling
- JavaScript – DOM manipulation, event handling, and LocalStorage



## 🔧 Installation & Usage
1. Clone the repo:
   git clone https://github.com/renea2024/recipe-manager.git
2. Open index.html in your browser
3. Start adding and managing your recipes
